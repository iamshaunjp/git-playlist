Welcome to Git & GitHub Totorial :-)

Enjoy it!

  1. Fork the project
  2. Clone the project to your local machine by typing the command:
    • git clone https://github.com/cyberclan/git-playlist.git
  3. make changes you want
  4. add changes to the branch
    • git add . or git add filename
  5. commit the changes with git commit -m "your comment"
  6. push the changes to master or wished branch
  7. Go to the website and create a new pull request you had done
